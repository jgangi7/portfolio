# Portfolio Website

My personal portfolio website showcasing my work and skills as a software developer.

## Quick Start

1. **Install**
   ```bash
   npm install
   ```

2. **Development**
   ```bash
   npm run dev
   ```
   Open `http://localhost:3000`

3. **Build & Deploy**
   ```bash
   npm run build
   npm start
   ```

## Features

- Responsive design
- Dark/Light mode
- Animated UI components
- Auto-hiding header

## Built With

- Next.js
- TypeScript
- TailwindCSS
- Framer Motion
